# Contributing to SMART GRADE

Thank you for considering contributing to SMART GRADE! 🎉

This guide will help you understand how you can contribute to this project.

---

## 📋 How Can I Contribute?

### Reporting Bugs

If you find a bug, please open an issue on GitHub with:

- What happened
- What you expected to happen
- Steps to reproduce the problem
- Your device and browser

### Suggesting Improvements

Have an idea for a new feature? Open an issue and explain:

- What you'd like to see
- Why it would be useful
- Any alternatives you've considered

---

## 📝 Making Changes

1. **Fork the repository** - Click the "Fork" button on GitHub
2. **Clone your fork** - `git clone https://github.com/your-username/SMART-GRADE.git`
3. **Create a branch** - `git checkout -b feature/your-feature-name`
4. **Make your changes**
5. **Commit your changes** - `git commit -m "Description of changes"`
6. **Push to your fork** - `git push origin feature/your-feature-name`
7. **Create a pull request** - Go to the original repository and click "New Pull Request"

---

## 🎨 Style Guidelines

- Use clear and descriptive variable names
- Keep code clean and well-organized
- Add comments for complex logic
