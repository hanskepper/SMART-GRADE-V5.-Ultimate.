1. System Overview

SMART GRADE is a Progressive Web Application (PWA) for grade management, designed for SIN GBHS FOUMBAN Form 5B Science. The application works 100% offline after first installation and stores all data locally on the user's device.

Key Features: 14 subjects, 6 sequences, 3 terms, 23 badges, 20 themes, 12 fonts, biometric login, QR code transfer, cloud backup, PWA offline support.
Data Privacy: All data stays on the user's device. No external servers store personal data unless the user explicitly uses cloud backup.

2. Technology Stack

HTML5

Semantic markup, localStorage API, Canvas API, MediaDevices API, Notification API, Vibration API, Clipboard API

CSS3

Flexbox, Grid Layout, CSS Animations, CSS Variables (Custom Properties), Glassmorphism (backdrop-filter), Media Queries, 20 Color Themes

External APIs

GitHub Gist API (cloud transfer), QR Code generation, Chart.js for graphs, Google Fonts, Font Awesome icons

3. Application Architecture

The application follows a modular architecture with separate JavaScript files for different concerns:

  • utils.js - Core utility functions (formatting, calculations, export/import)
  • database.js - All localStorage operations (CRUD for students, grades, subjects)
  • auth.js - PIN and fingerprint authentication
  • app.js - UI initialization (themes, menu, particles, notifications)
  • transfer.js - Cloud and QR code data transfer
  • confirm-dialog.js - Custom confirmation dialogs
  • install-handler.js - PWA installation across browsers
  • cloud-sync.js - Automatic cloud backup

7. Authentication System

The authentication system supports both PIN (4-digit) and fingerprint (WebAuthn) login methods.

// PIN Authentication var result = authenticateWithPin(studentId, pin); if (result.success) { // User is logged in as result.student // Redirect to dashboard } // Fingerprint Authentication loginWithFingerprint(function(result) { if (result.success) { // User authenticated via fingerprint // result.student contains user data } }); // Register Fingerprint registerFingerprint(studentId, studentName, function(result) { if (result.success) { // Fingerprint registered successfully } }); // Remove Fingerprint removeFingerprint(studentId); // Logout logout(); // Redirects to index.html
Fingerprint uses WebAuthn API and requires HTTPS or localhost. Not all browsers support this feature.

11. Progressive Web App (PWA)

The application is a fully functional PWA that works offline after first installation.

// Service Worker registration (in sw.js) self.addEventListener('install', function(event) { event.waitUntil( caches.open(CACHE_NAME).then(function(cache) { return cache.addAll(ASSETS); }) ); }); // Install handler (in install-handler.js) if (typeof InstallHandler !== 'undefined') { InstallHandler.init(); InstallHandler.install(); // Trigger installation } // Auto-update (in auto-updater.js) AutoUpdate.start(); // Checks GitHub every 10 minutes
The Service Worker caches all HTML, CSS, JS, and assets. The app works 100% offline after the first successful load.

12. Cloud Transfer System

The cloud transfer system uses GitHub Gist API (free, unlimited) to store temporary transfer data.

// Generate cloud transfer code CloudTransfer.generateCode(userId, userName, function(result) { if (result.success) { var code = result.code; // 6-character code // Share this code with the other device } }); // Import from cloud CloudTransfer.importWithCode(code, function(result) { if (result.success) { // Data imported successfully location.reload(); } }); // Generate QR Code QRTransfer.generateQR(code, 'containerId'); // Local transfer (same device/network) var localCode = generateLocalCode();
Important: Transfer codes expire after 10 minutes for security. QR codes are also valid for 10 minutes.

13. Export & Import System

The application supports multiple export formats: JSON (complete backup), CSV (spreadsheet), HTML (bulletin), PDF (report).

// Export all user data as JSON var jsonData = exportCompleteUserData(userId); // Download as file downloadJSON(jsonData, 'backup.json'); // Import JSON backup var result = importCompleteUserData(userId, jsonData); if (result.success) { // Data restored successfully } // CSV export format // Subject, Code, Sequence, Value, Date, Term // COMPUTER SCIENCES, CS, 1, 15.50, 2025-01-15, 1
The JSON backup includes all user data: grades, subjects, coefficients, achievements, streak, profile, flashcards, and history.
