Frequently Asked Questions

20+ answers
Account & Login
How do I recover my PIN?
For security reasons, PINs cannot be recovered. Your PIN is stored encrypted locally on your device. If you forget your PIN:
  • You will need to create a new account
  • You can restore your data from a backup if you have one
  • We strongly recommend saving your PIN in a secure place
Tip: Write down your PIN somewhere safe or use a password manager!
Fingerprint login not working?
  • Make sure your device has fingerprint hardware
  • You must first register your fingerprint in Settings
  • Check that you have granted fingerprint permission to your browser
  • Try re-registering your fingerprint in Settings
  • Use PIN login as an alternative
Tip: Fingerprint works best on Chrome and Edge browsers on Android.
Can I have multiple accounts?
Yes, you can create multiple accounts on the same device. Each student needs their own unique number within their class. The Home page displays all accounts for easy selection.
Data & Transfer
How do I transfer my data between devices?
Three methods available:
  • Cloud Transfer: Generate a cloud code, enter it on the other device
  • QR Code: Generate a QR code on source device, scan it on the destination
  • JSON File: Export as JSON file, transfer via USB/cloud
Tip: Cloud Transfer works between Chrome, Firefox, Safari, and Edge!
How do backups work?
  • Manual Backup: Click "Create Backup" in Backup page
  • Auto-Save: Grades are automatically saved every 30 seconds
  • Cloud Backup: Optional backup to JSONBin.io
  • All backups are stored locally and can be exported as JSON files
What data can I export?
You can export:
  • JSON: Complete backup (grades, badges, history, etc.)
  • CSV: Spreadsheet format for Excel
  • HTML Bulletin: Printable report card
  • PDF Bulletin: Professional PDF document
Technical Issues
App not loading / stuck?
  • Clear your browser cache and reload the page
  • Try reopening the app from home screen
  • Check if you have internet connection (first load requires internet)
  • Update your browser to the latest version
  • If problem persists, try reinstalling the PWA
Data not saving?
  • Check if localStorage is enabled in your browser
  • Make sure you have enough storage space on your device
  • Clear some space if localStorage is full (usually 5-10MB limit)
  • Try exporting your data as backup before clearing
  • Use the "Export JSON" feature to save your data
Tip: You can check storage usage in Developer Tools > Application > Storage.
QR code not scanning?
  • Allow camera permissions when prompted
  • Make sure you're in a well-lit environment
  • Hold the phone steady about 15-20cm from the QR code
  • Try using the alternative code entry method
  • Check that the QR code hasn't expired (10 minute validity)
Features
How to change theme or font?
Tap the Theme button in the top right corner. A bottom sheet will show:
  • 20 color themes - tap any to apply instantly
  • 12 font families - Inter, Roboto, Cinzel, and more
  • Night mode - automatic from 8 PM to 6 AM
  • Theme changes are saved automatically
How do I unlock badges?
23 badges available:
  • Streak badges: Use the app for 3, 7, 15, 30 consecutive days
  • Achievement badges: First Grade, Perfect Score, High Average, etc.
  • Special badges: Theme Collector, Font Collector, Photo Uploader
Tip: Check Achievements page to see your progress!
How do I set a goal?
  • On the Dashboard, use the Goal slider to set your target yearly average
  • In Goals page, you can set subject-specific targets using sliders
  • A progress bar shows how close you are to your target
  • You get a notification when you achieve your goal!
What are flashcards?
Flashcards help you study and memorize information:
  • Each subject has default flashcards (3-5 cards)
  • You can add your own custom flashcards
  • Tap the card to flip between question and answer
  • Navigate between cards with Prev/Next buttons
  • Unlock badges by creating 5 and 10 custom flashcards
Installation & PWA
How do I install the app?
Android (Chrome):
  • Open SMART GRADE in Chrome
  • Tap the menu button (3 dots) at top right
  • Select "Add to Home Screen" or "Install App"
  • Tap Install to add SMART GRADE to your home screen
iOS (Safari):
  • Open SMART GRADE in Safari
  • Tap the Share button at bottom
  • Scroll down and tap "Add to Home Screen"
  • Tap Add to install
Tip: Once installed, the app works 100% offline!
Does the app work offline?
Yes! SMART GRADE is a Progressive Web App (PWA) that works 100% offline after first installation:
  • All your data is stored locally on your device
  • You can add grades, view statistics, and use flashcards without internet
  • Only cloud backup and update checking require internet
  • The app will automatically sync when you're back online
